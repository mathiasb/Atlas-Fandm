A) Getting Started - top

In order to begin installing Atlas theme. You need the latest version of Wordpress. Here is the download URL. Once you downloaded and set it up, You are ready to go.

Installation

Upon downloading the package, you will have an "Atlas" folder inside the theme package. Upload this folder to your themes folder at: your WordPress installation root directory /wp-content/themes

Log into WordPress administration and click on the Appearance >> Themes tab. After that is done, activate the theme as you would activate any other theme.

Note: You have to set /cache folder of Atlas writable (chmod 777)


Setup Main Menu

Atlas has 1 menus locations, the menu align under the logo.

Go to Appearance > Menus then click "Create Menu" name it "Main Menu". Add items to your menu and select Theme location.



For those who are not familiar with Wordpress 3.0 menu below are some tutorials.

  1. http://templatic.com/news/wordpress-3-0-menu-management
  2. http://www.wonderhowto.com/how-to-use-new-menu-system-wordpress-3-0-thelonious-376792/

Theme Admin Panel

Upon activating the theme, from your WordPress admin panel, you will notice the "Atlas" options tab at the bottom of your WordPress admin panel menu options. Click on it. You will open the admin panel of Atlas.


How to add a new gallery?

  1. Open Galleries > Add New
  2. Enter gallery's name and description
  3. Publish it


How to add a gallery's image?

  1. Open gallery page
  2. Click on Gallery Images > Add New button
  3. Select files to upload (you can use SHIFT or CTRL for multiple images uploading)
  4. Once the upload progress is completed. It will automatically add to gallery


How to add a gallery page?

  1. Click on "Pages > Add New"
  2. Enter page title
  3. On Page Attributes box, select template which has gallery keyword

  4. There is "Attach Gallery" option, select which gallery you want to display on the page
  5. Click on "Publish" to make this page viewable.


How to add a password protected gallery page?

  1. Click on "Pages > Add New"
  2. Enter page title
  3. On Page Attributes box, select template which has gallery keyword
  4. There is Password Protect option, enter your gallery password there
  5. Click on "Publish" to make this page viewable.


How to add other portfolio content ex. video, links?

  1. Click on "Portfolios > Add New Portfolio"
  2. Enter title, its content
  3. Click on "Set featured image" to upload preview image for this post
  4. Once uploaded click "Use as featured image"
  5. There is "Content Type" option, you can select from image, video or external link
  6. Click on "Publish" to make this post viewable.


How to categorize portfolio items to a set?

To use portfolio sets.
Once you assign portfolio to set. Go to appearance >menus. Then select which set you want to add to your menu items.

If you can't see it, click on screen option (top right) and check on sets


How to select gallery for homepage

Open Atlas admin > Homepage and Select homepage gallery. You can also change homepage gallery style there.


How to change gallery and homepage images order?

Open Atlas admin > Galleries > Gallery sort by and Atlas admin > Homepage > Homepage Gallery sort by


How to add a logo image to homepage?

Open Atlas admin > Homepage and Select image for homepage logo


How to add a background image to blog page?

Open Atlas admin > Blog-Portfolio and Select image for blog background (Recommended size 1440x900 pixels)


How to add a background image to portfolio page?

Open Atlas admin > Blog-Portfolio and Select image for portfolio background (Recommended size 1440x900 pixels)


How to add a background image to other page?

  1. Open edit page
  2. Click on "Set featured image" to upload preview image for this post
  3. Once uploaded click "Use as featured image"
  4. Click on "Publish" to make this post viewable.


How to add a blog page?

  1. Click on "Pages > Add New"
  2. Enter page title
  3. On Page Attributes box, select template as "Blog"
  4. Click on "Publish" to make this page viewable.


How to add a blog post?

  1. Click on "Posts > Add New"
  2. Enter title, its content
  3. (Optional) Select the post category
  4. Click on "Set featured image" to upload preview image for this post
  5. Once uploaded click "Use as featured image"
  6. Click on "Publish" to make this post viewable.


How to add a contact page?

  1. Click on "Pages > Add New"
  2. Enter page title
  3. On Page Attributes box, select template as "Contact"
  4. Click on "Publish" to make this page viewable.

Then go to Atlas Admin Panel > General and setup your email address.

How to add your own header fonts?

  1. Prepare your font file. You can also find free fonts from http://fontsquirrel.com/
  2. Go to Cufon site to generate font .js file
  3. Open Thinker admin > Font > Header Font Style
  4. Upload your font and save changes.

How to change site's elements colors?

  1. Go to Atlas Admin Panel > Colors.
  2. Choose the color from color picker for each element


Atlas custom widgets

However the homepage sidebar is support all Wordpress 3.0 widgets. I created 8 custom widgets to make your website more interesting.

You can also add your sidebar via Admin panel.

To add content to sidebar simple go to Appearance > Widgets and drag item into certain sidebar.


Sidebar widget with shortcode support

You can also add shortcode to sidebar widget, simply select "Text" type for your widget and enter shortcode to it.

Shortcode Generator

Atlas also come with built-in plugin called "Shortcode Generator" which will help you create theme's shortcode from GUI. Simply click on "Shortcode Generator" once you activate the theme.

Select shortcode type you want to add then enter its contents. Click on "Generate Shortcode" and copy all code inside Shortcode box. Go to your post or page paste the code to its content.


Troubleshooting

Atlas also come with built-in plugin called "Troubleshooting" which will help you solve common issues that might arise from different server settings.

B) CSS Files and Structure - top

I'm using five CSS files in this theme. The first file contains all of the specific stylings for the page. The file is separated into sections using:

	
	/* CSS Reset */

	some code

	/* Global layout */
	
	some code
	
	/* Input layout */
	
	some code


The second file is a styling for jQuery UI element.

The third file is a styling for modal window.

The fourth and fifth file is a styling for HTML5 video.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

C) JavaScript - top

This theme imports eight Javascript files.

  1. jQuery
  2. jQuery UI
  3. jQuery Validate
  4. jQuery fancybox
  5. jQuery nivoslider
  6. jQuery easing
  7. Browser check
  8. Custom script
  1. jQuery and jQuery UI are a Javascript library that greatly reduces the amount of code that you must write.
  2. jQuery validate contact form
  3. jQuery Fancybox create a nice popup window using in gallery
  4. jQuery nivoslider for homepage slider
  5. jQuery easing for slider transition effect
  6. Browser check check browser name and version
  7. Most of the animation in this theme is carried out from the customs scripts.
